 Workflow Automation in Healthcare Dashboard

Project Overview
This Tableau dashboard analyzes the impact of workflow automation across hospital departments, focusing on time saved, error reduction, staff satisfaction, and cost savings.

 Files Included
- `healthcare_automation.csv` – Sample dataset
- `healthcare_automation.twbx` – Tableau packaged workbook
- `dashboard_screenshot.png` – Preview of the dashboard

 Key Insights
- Radiology and IT departments save the most time
- Error rates drop by 70–80% after automation
- Staff satisfaction increases with higher adoption rates

 How to Use
1. Download the `.twbx` file
2. Open in Tableau Desktop or Tableau Public
3. Explore the interactive dashboard
